# 5.2.2. 브레이크 해제 스위치

브레이크 해제유닛의 스위치 배치는 그림 5.3과 같으며 각각에 대한 용도 및 동작설명은 표 5-4와 같습니다. 원하는 축의 브레이크를 해제하기 위해서는 먼저 Enable 버튼을 누르고, Enable 버튼을 누르고 있는 상태에서 B1~B8 중 버튼을 동시에 누르면 해당 축이 해제됩니다.
